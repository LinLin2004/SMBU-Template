# SMBU 本科生毕业设计 LaTeX 模板

本模板专为**深圳北理莫斯科大学（SMBU）** 本科生毕业设计（论文）撰写而设计。它基于标准 LaTeX 编写，能够帮助同学们快速生成符合学校格式要求的毕业论文。

模板从目录开始（目录会自动生成）后面内容正常填写即可。
封面和最后表格实属难做，可使用其中的word将个人信息填入然后变成pdf使用命令插入即可。latex源码中已经明确给了示例表示。

**请使用XeLaTeX编译！**
